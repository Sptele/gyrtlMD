# GyrtlMD

A simple multiplier (mul.py) and divider (div.py). Includes a simple testbed that multiplies/divides all the ordered pairs in the cartesian product 100 x 100 (for division, this excludes all cases
where the divisor would be zero).

Developed by Gautam Khajuria for CMPTGCS 1B W26 under Prof. Balkind.
